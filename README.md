`riskspatial` is an R package for applying property & casualty (P&C) insurance loss
models directly to spatial exposure data. It bridges geospatial analysis (`sf`)
with insurance concepts such as exposure, intensity, vulnerability, and loss.

Unlike traditional actuarial packages, `riskspatial` treats geography as a
first-class component of risk.
